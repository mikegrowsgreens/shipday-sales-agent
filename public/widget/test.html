SalesHub Chat Widget - Test Page

This page demonstrates the embeddable chat widget. Look for the chat bubble in the bottom-right corner.

How to embed on your site

Add this single script tag before the closing </body> tag:

<script
  src="https://your-saleshub.com/widget/embed.js"
  data-org-slug="your-org"
  data-accent-color="#2563eb"
  data-position="bottom-right"
  data-greeting="Hi! How can I help?"
></script>

Configuration Options

data-org-slug — Your organization slug (required)

data-accent-color — Hex color for the widget theme (default: #2563eb)

data-position — "bottom-right" or "bottom-left" (default: bottom-right)

data-greeting — Custom greeting message

data-api-base — API base URL (auto-detected from script src)

Iframe Fallback

For sites that can't use script tags, use the iframe embed:

<iframe
  src="https://your-saleshub.com/widget/embed/your-org"
  style="position:fixed; bottom:0; right:0;
         width:400px; height:600px;
         border:none; z-index:999999;"
  allow="clipboard-write"
></iframe>

Features

  • Shadow DOM — no CSS conflicts with your site
  • Session persistence via localStorage (24h expiry)
  • Mobile-responsive — full-screen on mobile, widget on desktop
  • Visitor tracking — captures page URL, referrer, UTM params
  • Smooth open/close animations
  • Suggested prompts after each response
  • Auto-detects API base URL from script source
