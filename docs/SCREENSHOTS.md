### How to Capture

**Option A: Browser DevTools**
1. Open Chrome DevTools (Cmd+Opt+I)
2. Toggle device toolbar (Cmd+Shift+M)
3. Set dimensions: 1280x800 for desktop, 390x844 for mobile
4. Use "Capture screenshot" from the three-dot menu in DevTools

**Option B: macOS Screenshot**
1. Cmd+Shift+4 for selection
2. Cmd+Shift+5 for window capture
